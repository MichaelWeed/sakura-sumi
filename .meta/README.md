# Project Management Files

This directory contains project management and meta-documentation files that are not part of the deployable package.

## Structure

- **CANON.md** - Immutable project rules and specifications
- **NOTES.MD** - High-level project overview and status
- **WORKING.md** - Active tasks and working notes
- **notes.yaml** - Documentation registry
- **bugs/** - Bug tracking YAML files
- **updates/** - Epic and backlog YAML files
- **archive/** - Historical records

## Purpose

These files are for project management, development workflow, and historical tracking. They are excluded from deployment packages and distribution builds.
